## github常用命令
 git init   把一个普通仓库交给git 管理
 git status  用于显示工作目录和暂存区的状态
 git commit -m  “描述提交的文件” 提交
 git pull 将本地版本库的分支推送到远程服务器上对应的分支了,
 git add .  添加当前目录的所有文件到栈存里面去
 git commit --amend --no-edit    把刚才变化的文件追加上去
